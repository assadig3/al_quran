# 📖 تطبيق القرآن الكريم | Quran App

تطبيق قرآني مميز يهدف إلى تسهيل قراءة القرآن الكريم وتصفح السور بطريقة سلسة واحترافية باستخدام واجهة حديثة تعتمد على Jetpack Compose.

---

## ✨ الميزات الرئيسية

- عرض سور القرآن الكريم بتصميم أنيق
- دعم الوضع الليلي (Dark Mode)
- تنقل سريع بين السور والآيات
- توافق مع معظم إصدارات Android
- يعتمد على Android Jetpack و Kotlin

---

## 🚀 البدء

### 1. المتطلبات:
- Android Studio (أحدث إصدار)
- Kotlin 1.9+
- Gradle 8.2+
- SDK 36

git clone https://github.com/assadig3/al_quran.git
